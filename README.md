# run
Added new script for running compiled programs.
Usage:
```bash
run main
```
Added new feature, now you can whatch your program runtime just typing:
```bash
run main -t
```

Installing:


```bash
sudo cp compilecpp /bin/compilecpp
sudo cp run /bin/run
```
